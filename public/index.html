CrackedPulse

CrackedPulse Launcher

Маленький Windows-лаунчер для локальної збірки Minecraft 1.21.4.

GitHub Інструкція

Як запустити

  1. Збери `crackedpulse.exe` через GitHub Actions або `npm run dist:win`.
  2. Поклади папку `CrackedPulse` на ПК.
  3. Відкрий налаштування в лаунчері й вибери папку з `loader-1.21.4.ps1`.
  4. Введи нік і натисни `Грати`.
